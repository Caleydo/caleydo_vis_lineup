=====================

Phovea wrapper for [LineUp.js](http://github.com/Caleydo/lineup.js) to visualize tables as ranking.

Installation
------------

```
git clone https://github.com/phovea/phovea_vis_lineup.git
cd phovea_vis_lineup
npm install
```

Testing
-------

```
npm run test
```

Building
--------

```
npm run build
```
